# MTVerse — Modern Admin Dashboard

A modern, fully-featured admin dashboard built with Next.js 16, React 19, Tailwind CSS v4, and shadcn/ui.

## Getting Started

1. Install Dependencies
```bash
npm install
```

2. Run the development server
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm run start
```

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript 5
- **Styling**: Tailwind CSS 4
- **UI Components**: shadcn/ui (New York style)
- **Charts**: ApexCharts
- **Tables**: TanStack Table v8
- **Icons**: Iconify (Solar, Tabler, Lucide)
- **Theme**: next-themes (light/dark)

## Features

- 96+ pages across UI Kit, Charts, Forms, Tables, Apps, Pages, Auth, Widgets, Dashboards
- Full light/dark mode with system preference detection
- Responsive design — mobile, tablet, desktop
- Accessible components with ARIA support
- Production-ready, statically generated
